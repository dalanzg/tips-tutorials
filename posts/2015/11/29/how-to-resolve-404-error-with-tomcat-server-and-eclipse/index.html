How to resolve 404 error with Tomcat Server and Eclipse

Published on November 29, 2015 by Daniel Lanza

 java

If you start Tomcat from Eclipse and get a 404-Error code when you go to http://localhost:8080, there is a easy way to fix the issue.

Just go to Properties for Tomcat (right click on Tomcat - Servers Tab)

Switch location from [workspace metadata] to [/Servers/Tomcat v…]

Restart Tomcat from Eclipse and go back to http://localhost:8080. You will see Tomcat configuration page.
